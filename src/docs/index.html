API Documentation

/api/login

POST

Description: User login

Request Body:

{
    "name": "string",
    "password": "string"
}

Response:

  • 200: Login successful, sets session cookie
  • 400: Bad request or already logged in
  • 401: Unauthorized
  • 404: Not found

/api/logout

POST

Description: User logout

Response:

  • 200: Logout successful, clears session cookie
  • 400: Not logged in
  • 404: Not found

/api/user

GET

Description: Get user information

Request Body:

{
    "name": "string",
    "id": "number"
}

Response:

  • 200: Returns user information
  • 400: Bad request
  • 404: Not found

POST

Description: Create new user

Request Body:

{
    "name": "string",
    "password": "string"
}

Response:

  • 201: Created successfully
  • 400: User already exists or bad request

PUT

Description: Update user password

Request Body:

{
    "name": "string",
    "password": "string"
}

Response:

  • 200: Updated successfully
  • 400: Bad request
  • 401: Unauthorized
  • 404: Not found

/api/token

POST

Description: Create new token

Response:

  • 201: Created successfully, returns token
  • 401: Unauthorized
  • 404: Not found

/api/rooms

GET

Description: Get room information

Request Body:

{
    "room_id": "number",
    "game_id": "number"
}

Response:

  • 200: Returns room information
  • 400: Bad request
  • 404: Not found

POST

Description: Create or join room

Request Body:

{
    "player_id": "number",
    "game_id": "number"
}

Response:

  • 201: Created successfully
  • 200: Joined successfully
  • 400: Bad request or already in a room
  • 401: Unauthorized
  • 404: Not found

DELETE

Description: Delete room

Request Body:

{
    "room_id": "number"
}

Response:

  • 200: Deleted successfully
  • 400: Bad request
  • 401: Unauthorized
  • 404: Not found

POST

Description: Make a move in the game

Request Body:

{
    "column": "number"
}
